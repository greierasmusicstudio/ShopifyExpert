# Shopify Expert Landing Page

Site de prezentare pentru servicii de creare magazine online pe Shopify.

## Structura proiectului

- `index.html` - Pagina principală
- `style.css` - Stilurile CSS
- `script.js` - JavaScript pentru funcționalități interactive
- `images/` - Director pentru imagini (logo, background, etc.)

## Caracteristici

1. **Design responsive** - Se adaptează la toate dispozitivele
2. **3 pachete de servicii** - Basic, Plus, Top
3. **Formular de contact** - Trimite cereri de ofertă
4. **Comparație detaliată** - Tabel comparativ între pachete
5. **WhatsApp integration** - Buton flotant pentru contact rapid
6. **Conformitate ANPC/SOL** - Link-uri către autorități

## Cum să rulezi

1. Clonează repository-ul
2. Deschide `index.html` în browser
3. Pentru hosting, încarcă toate fișierele pe GitHub Pages sau orice hosting

## Personalizare

1. **Culori**: Modifică variabilele CSS din `:root` în `style.css`
2. **Prețuri**: Actualizează în `index.html` în secțiunea pachetelor
3. **Contact**: Actualizează numerele de telefon și email
4. **Imagini**: Înlocuiește imaginile din directorul `images/`

## Servicii integrate

- **Formspree** pentru formular (configurabil)
- **Font Awesome** pentru iconițe
- **Google Fonts** pentru tipografie

Acest site este creat pentru a promova servicii de creare magazine Shopify. Asigură-te că ai permisiunea de a utiliza logo-ul Shopify și că menționezi că nu ești afiliat cu Shopify Inc.
